## Installation

Clone the project

```bash
  git clone https://github.com/shubhanshu1911/Webnotes_fontend.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
